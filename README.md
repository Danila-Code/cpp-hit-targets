## Описание
Приложение определяет оптимальные координаты центра окружности заданного радиуса и количество точек на плоскости, которые располагаются в её пределах. Другими словами, программа рассчитывает оптимальные координаты запуска ракеты с заданным радиусом поражения, чтобы поразить как можно больше целей.

Приложение имеет консольный интерфейс. При запуске программы задаются параметры:
* текстовый файл с координатами целей
* радиус (поражения)

## Технологии
* C++ 20
* STL
* ООП
* quad tree
* GTest
* conan
* CMake

## Особенности
* хранение точек (целей) на плоскости реализовано с помощью дерева квадрантов
* каждый узел дерева хранит размеры прямоугольной области на плоскости, четыре точки и четыре указателя на подузлы
* дерево квадрантов помогает эффективно находить координаты центра окружности, охватывающей максимальное количество точек
* сложность вставки нового узла в дерево O(logn)
* сложность поиска оптимальной точки O(k*l*(logn + m)),
где k и l соответственн размеры области поиска, а m - количество найденных точек

## Сборка
1. Скачайте файлы из текущего репозитория.
2. В терминале в папке cpp-hit-targets введите:
	`conan install . --output-folder=build --build=missing -pr default`
3. Создайте в корневой папке проекта папку "build" для сборки проекта, перейдите в неё в командной строке и запустите cmake (для включения/выключения опции сборки тестов используйте флаг `-DBUILD_TESTS`, по умолчанию тесты собираются):\
	`cmake -S . -B build2 -DCMAKE_TOOLCHAIN_FILE=build2/build/generators/conan_toolchain.cmake -DBUILD_TESTS=ON`\
4. Запустите сборку проекта в командной строке:\
	`cmake --build build --config Release`\
*Проект собран.*

## Запуск
* Чтобы запустить программу нужно в командной строке, находясь в папке `build` проекта, ввести:\
	`Release/hit_targets.exe <"входной файл данных txt"> <радиус>`
* Для запуска тестов (если проект собирался с флагом `-DBUILD_TESTS=ON`) нужно в терминале, находясь в папке `build`, ввести:\
	`tests/Release/tests.exe`\

## Формат файла с координатами точек (целей)
Входной файл имеет формат txt, в котором на каждой строчке располагается пара координта x и y - целых чисел, разделённых запятой, например:
```
10,2
45,16
23,18
```

## Системные требования
Компилятор С++, С++20, GTest 1.17.0 (опционально), conan (для GTest) CMake 3.8
